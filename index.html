About the project:

The main goal was to build a 3d maze by using the knowledge we claimed during the course. At the beginning we learned how oop works in java , we learnd about inheritance , abstract classes and object classes.

Teachnical information:

The Project contains several extensive design patterns including Stratagy, Decorator, Command and more. The architecture based on MVP and MVC, and the algorithms which took place were BFS and DFS. The application contains both Graphic user interface and and Command line interface- depands on user decision each. For the Graphic interface based on SWT libraries. There is use of Input and output streams, and XML decoder and encoder for saving the current game properties to local storage. For efficency we decided to run the application on Threads, and the count of threads depands on user decision.
